# Garden Advice App

The Garden Advice App is a simple JavaScript program designed for gardening enthusiasts.  
It provides basic gardening tips based on the **season** and **type of plant** entered by the user.

This project was created to demonstrate good **Git workflows**, **code refactoring**, and **documentation practices**.

---

## Project Features

- Prompts the user for:
  - Current season
  - Plant type
- Displays gardening advice in the browser console
- Uses reusable functions for clean and readable code
- Includes detailed comments and documentation

---

## Technologies Used

- JavaScript (ES6)
- HTML
- Git & GitHub

## How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/faithze/garden-app.git
````

2. Open the project folder:

   ```bash
   cd garden-app
   ```

3. Open `index.html` in your web browser.

4. Enter the requested inputs when prompted.

5. View the gardening advice in the **browser console**.

## Git Workflow Used

This project follows a structured GitHub workflow:

* GitHub Issues were created to track improvements
* Separate branches were used for each issue
* Changes were committed with descriptive messages
* Pull Requests were opened, reviewed, and merged
* Issues were closed once completed

This mirrors real-world collaborative development practices.
